# 一位数字自由人的自我介绍 :blush:

### **你好，我是 星X箭（all-in-one) 👋**

这里是我记录思考、技术实验与生活碎片的地方。如果用几个关键词来定义我，那大概是：**架构师 / 量化交易者 / 数字化玄学研究员**。

- **💻 生产力工具控** 我享受在 macOS 与 Windows 之间丝滑同步 P.A.R.A. 目录结构的快感。我更倾向于使用 Markdown 记录一切，并配合 Hugo 与 SQLite 这种轻量级方案来展示我的项目进度。
- **🤖 AI 实验者** 我热衷于探索 AI 的边界，从编写自定义的 Claude Code 技能到开发 八字命理 服务，我始终相信 AI 应该作为人类智力的外挂而存在。
- **📈 概率玩家** 在 A 股市场，我通过 QMT 和 TradingView 寻找量化信号；在代码世界，我用 Go 语言重构算法。无论在哪个领域，我都在寻找那条“最优路径”。
- **⛩️ 传统与未来的交汇** 你可能会看到我前一秒在处理企业级 ERP 的复杂逻辑，后一秒在用儒略日（Julian Day）推算未来年的干支。对我来说，代码与古籍同样迷人。
